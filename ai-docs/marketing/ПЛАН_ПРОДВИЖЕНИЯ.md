# План продвижения DotFlat

**Дата**: 2026-05-02

---

## Главная проблема

$5k TVL → $20k — это не маркетинговая задача, а задача доверия.  
Без имени в агрегаторах, без видимости в сообществе — любой разговор заканчивается "а вы где?"

**Логика**: видимость → контакты → пользователи. Не наоборот.

---

## Шаг 0: Листинги (бесплатно, без контактов, сделать первым)

| Платформа | Что делать |
|---|---|
| **DeFiLlama** | Сабмит через GitHub PR в их репо (`DefiLlama/DefiLlama-Adapters`) |
| **DeBank** | Создать профиль проекта |
| **CoinGecko** | Форма для добавления токена RLE |

Без этого: любой потенциальный пользователь гуглит название и видит пустоту.  
С этим: базовая легитимность, которая работает фоном.

## Шаг 1: Русскоязычные Telegram

Telegram-группы с реальными людьми, у которых есть деньги и интерес к DeFi:

| Группа | Аудитория |
|---|---|
| `@defi_ru` | Основная русская DeFi тусовка |
| `@ethereum_ru` | Ethereum-девелоперы и энтузиасты |
| `@web3_ru` | Строители протоколов |
| `@defichat_ru` | DeFi-пользователи |

**Правило входа**: не "зайти и запостить ссылку". Зайти, посидеть неделю, ответить на несколько вопросов по DeFi-механике, потом — когда уместно — упомянуть проект. Это занимает 15-20 минут в день.

---

## Шаг 2: Gitcoin Grants

Деньги без контактов — только через техническое описание и публичное голосование.

**Что нужно**: описание проекта на английском + GitHub с кодом + базовая документация.  
**Дедлайн**: следить на gitcoin.co за датами следующего раунда.

Кроме денег: тысячи человек видят проект при голосовании. Часть — потенциальные пользователи.

---

## Шаг 3: Один технический тред на Twitter/X

Не "мы запустили", а объяснение конкретного механизма:

> "Почему commit-reveal auction лучше для ликвидаций чем флэш-ботный аукцион. На реальном примере нашего протокола."

Или:

> "Почему привязка к корзине 33 commodities защищает от инфляции, а USDC — нет. Данные за 5 лет."

DeFi Twitter маленький и технически вовлечённый. Один хороший тред читают 500-1000 нужных людей. Ответы на комментарии — это новые контакты.

---

## Шаг 4: ETHGlobal онлайн-хакатоны

- Участие онлайн, без поездки
- Можно подавать существующий протокол с надстройкой (новая фича, интеграция)
- Призы = деньги + публичность + контакты с джаджами (они часто нанимают или рекомендуют)

Расписание: ethglobal.com

## Нарратив для всех каналов

> **DAI и USDC теряют покупательную способность вместе с долларом.  
> DotFlat привязан к 33 реальным товарам — нефти, золоту, пшенице, меди.  
> Инфляция доллара не съедает твои сбережения.**

Целевая аудитория: криптопользователи, которые понимают инфляцию и ищут защиту покупательной способности. Не трейдеры — долгосрочные держатели.
