Flash Minting (Week 20)
~~~~~~~~~~~~~~~~~~~~~~~~

What is Flash Minting::

    Borrow DFC for 1 transaction:
    ├─ Mint DFC at start of tx
    ├─ Use it however you want
    ├─ Return it + fee by end of tx
    ├─ If not returned, tx reverts
    └─ Zero collateral needed!
    
    Fee: 0.05% of minted amount

Use Cases::

    1. Arbitrage:
       ├─ Mint 1M DFC
       ├─ Buy on DEX at 0.98
       ├─ Sell on CEX at 1.0
       ├─ Return 1M DFC + $500 fee
       └─ Keep $20k profit
    
    2. Collateral Swapping:
       ├─ Flash mint DFC
       ├─ Repay CDP debt
       ├─ Withdraw old collateral
       ├─ Deposit new collateral
       ├─ Mint new debt
       └─ Return flash mint
    
    3. Liquidation Protection:
       ├─ Flash mint DFC
       ├─ Repay debt
       ├─ Add more collateral
       ├─ Re-mint debt
       └─ Avoid liquidation

Benefits::

    For Protocol:
    ├─ Tighter price peg ($0.98-1.02)
    ├─ Additional fee revenue
    ├─ Capital efficiency showcase
    └─ Attracts sophisticated users
    
    For Users:
    ├─ Arbitrage opportunities
    ├─ Advanced strategies
    ├─ Lower liquidation risk
    └─ Better capital efficiency
