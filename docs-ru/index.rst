Документация Dotflat
=====================

Эта документация предназначена для того, чтобы помочь вам начать работу с Dotflat.
Она охватывает Dotflat как концепцию, объясняет основные способы использования Dotflat, более подробную информацию о смарт-контрактах системы и продвинутые темы для более сложных приложений и случаев использования.
