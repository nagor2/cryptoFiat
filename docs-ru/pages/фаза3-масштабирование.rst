Флеш-минтинг (Неделя 20)
~~~~~~~~~~~~~~~~~~~~~~~~~

Что такое флеш-минтинг::

    Заем DFC на 1 транзакцию:
    ├─ Минт DFC в начале транзакции
    ├─ Используй как хочешь
    ├─ Верни + комиссию к концу транзакции
    ├─ Если не вернул, транзакция откатывается
    └─ Нулевой залог нужен!
    
    Комиссия: 0.05% от заминченной суммы

Use cases::

    1. Арбитраж:
       ├─ Минт 1M DFC
       ├─ Купи на DEX по 0.98
       ├─ Продай на CEX по 1.0
       ├─ Верни 1M DFC + $500 комиссия
       └─ Оставь $20k профита
    
    2. Своп залога:
       ├─ Флеш-минт DFC
       ├─ Погаси долг CDP
       ├─ Выведи старый залог
       ├─ Внеси новый залог
       ├─ Минт новый долг
       └─ Верни флеш-минт
    
    3. Защита от ликвидации:
       ├─ Флеш-минт DFC
       ├─ Погаси долг
       ├─ Добавь больше залога
       ├─ Перемити долг
       └─ Избеги ликвидации

Преимущества::

    Для протокола:
    ├─ Более узкая привязка цены ($0.98-1.02)
    ├─ Дополнительный доход от комиссий
    ├─ Демонстрация эффективности капитала
    └─ Привлечение продвинутых пользователей
    
    Для пользователей:
    ├─ Возможности арбитража
    ├─ Продвинутые стратегии
    ├─ Меньший риск ликвидации
    └─ Лучшая эффективность капитала

Governance V2 (Неделя 22)
~~~~~~~~~~~~~~~~~~~~~~~~~~

Улучшения над V1::

    1. Делегирование:
       ├─ Делегирование силы голоса
       ├─ Токены остаются в кошельке
       ├─ Профессиональные делегаты
       └─ Более высокое участие
    
    2. Vote Escrowed RULE (vRULE):
       ├─ Блокировка RULE → получение vRULE
       ├─ Дольше блокировка = больше силы
       ├─ Распределение доходов
       └─ Выровненные стимулы
    
    3. Интеграция Snapshot:
       ├─ Off-chain голосование (без газа)
       ├─ Сложные стратегии голосования
       ├─ Лучший UX
       └─ Более высокое участие
    
    4. Множественные типы предложений:
       ├─ Изменения параметров
       ├─ Добавление функций
       ├─ Распределение бюджета
       ├─ Стратегические решения
       └─ Инициативы сообщества

Ожидаемое влияние::

    Участие в управлении:
    ├─ V1: 5% от предложения голосует
