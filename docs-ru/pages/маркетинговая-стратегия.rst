Маркетинговая стратегия
=======================

Обзор
-----

Этот документ описывает комплексную маркетинговую стратегию и стратегию роста проекта Dotflat (DFC) - первого стейблкоина, обеспеченного сырьевыми товарами, на Ethereum.

Уникальное ценностное предложение
----------------------------------

Ключевое позиционирование
~~~~~~~~~~~~~~~~~~~~~~~~~

**Главное конкурентное преимущество**: DFC обеспечен корзиной сырьевых товаров, а не USD, обеспечивая:

* 📊 **"Стейблкоин, защищенный от фиатной инфляции"**
* 🏆 **"Цифровое золото 2.0 с корзиной сырьевых товаров"**
* 🌍 **"Глобальная валюта, независимая от центральных банков"**

Почему это важно
~~~~~~~~~~~~~~~~~

Традиционные стейблкоины (USDT, USDC, DAI):
- Подвержены USD инфляции
- Зависят от банковской системы
- Подвержены регуляторным рискам
- Централизованы (кроме DAI)

DFC предлагает:
- Обеспечение реальными активами (золото, медь, нефть и т.д.)
- Децентрализованное управление через токены RULE
- Защита от обесценения фиатной валюты
- Прозрачные резервы on-chain

1. Bootstrap ликвидности
~~~~~~~~~~~~~~~~~~~~~~~~~

**Критический фактор успеха**: Без ликвидности проект умирает.

Программа Liquidity Mining
^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Общее размещение**: 250,000 токенов RULE на 6 месяцев

Распределение по пулам::

    DFC/ETH     → 40% (100,000 RULE)
    DFC/USDC    → 30% (75,000 RULE)
    RULE/ETH    → 20% (50,000 RULE)
    RULE/DFC    → 10% (25,000 RULE)

**Целевой APY**: 200-500% на запуске, постепенное снижение

Ожидаемые результаты:

* Месяц 1: $2-5M ликвидности
* Месяц 2: $10-20M ликвидности
* Месяц 3: $30-50M ликвидности
* Месяц 6: $100M+ ликвидности

Стимулы открытия CDP
^^^^^^^^^^^^^^^^^^^^^

Награждение пользователей за открытие CDP позиций::

    Первые 100 CDP     → 100 RULE каждый (10,000 RULE всего)
    Следующие 400 CDP  → 50 RULE каждый (20,000 RULE всего)
    Следующие 1,000 CDP → 25 RULE каждый (25,000 RULE всего)
    
    Бонусные стимулы:
    - CDP > 10 ETH     → +50 RULE бонус
    - CDP > 50 ETH     → +200 RULE бонус
    - CDP > 100 ETH    → +500 RULE бонус

**Общее размещение**: 55,000 RULE + бонусы (~20,000 RULE) = 75,000 RULE

Ожидаемые результаты:

* 1,500 открытых CDP позиций
* $50-100M TVL только от CDP
* Сильная вовлеченность протокола

Vote-Escrowed токеномика (vRULE)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Вдохновлено моделью veCRV Curve**::

    Заблокировать RULE → Получить vRULE (голосующую силу)
    
    Варианты длительности блокировки:
    ├─ 1 неделя  → 1.00 RULE = 0.01 vRULE
    ├─ 1 месяц   → 1.00 RULE = 0.04 vRULE
    ├─ 3 месяца  → 1.00 RULE = 0.12 vRULE
    ├─ 6 месяцев → 1.00 RULE = 0.25 vRULE
    ├─ 1 год     → 1.00 RULE = 0.50 vRULE
    ├─ 2 года    → 1.00 RULE = 0.75 vRULE
    └─ 4 года    → 1.00 RULE = 1.00 vRULE (макс)

**Преимущества vRULE**:

1. **Голосование в управлении**: 1 vRULE = 1 голос
2. **Распределение комиссий**: Доля от дохода протокола
3. **Boost LP наград**: До 2.5x множителя
4. **Приоритетный доступ**: Новые функции, партнерства

**Распределение комиссий**:

Все комиссии протокола собираются в DFC::

    Комиссии процентов CDP  → 1% от всех позиций
    Штрафы за ликвидацию    → 13% от ликвидированной суммы
    Комиссии раннего вывода → 0.5% от ранних выводов

    100% распределяется держателям vRULE еженедельно

**Ожидаемый APY для держателей vRULE**: 15-30% (зависит от комиссий)

12. Митигация рисков
~~~~~~~~~~~~~~~~~~~~~

Выявленные риски и решения
^^^^^^^^^^^^^^^^^^^^^^^^^^^

**1. Риск оракула**

Проблема: Манипуляция или отказ price feed

Решения::

    ├─ Использовать Chainlink (наиболее безопасный)
    ├─ Реализовать fallback оракулы (Band Protocol)
    ├─ Time-weighted average prices (TWAP)
    ├─ Circuit breakers для крупных ценовых движений
    └─ Множественные источники данных на сырьевой товар

**2. Риск смарт-контракта**

Проблема: Баги, ведущие к потере средств

Решения::

    ├─ 3+ профессиональных аудита
    ├─ $500k программа bug bounty
    ├─ Постепенный rollout с TVL лимитами
    ├─ Страховое покрытие
    ├─ Функционал экстренной паузы
    └─ Регулярные аудиты кода

**3. Риск ликвидности**

Проблема: Невозможность выйти из крупных позиций

Решения::

    ├─ Стимулы liquidity mining
    ├─ Пул Curve для глубокой ликвидности
    ├─ Партнерства с маркет-мейкерами
    ├─ Постепенное снижение стимулов (не обрывом)
    └─ Множественные торговые площадки

**4. Регуляторный риск**

Проблема: Регулирование стейблкоинов

Решения::

    ├─ Полностью децентрализованный (нет центрального эмитента)
    ├─ Структура управления DAO
    ├─ Подготовленные юридические заключения
    ├─ Нет зависимостей от US entity
    ├─ Прозрачные on-chain операции
    └─ Не позиционируется как "инвестиция"

**5. Риск спирали смерти**

Проблема: Bank run на DFC, каскад ликвидаций

Решения::

    ├─ Высокая коллатерализация (170%)
    ├─ Аукционы ликвидации (не мгновенные)
    ├─ Стабилизационный фонд (5% от supply)
    ├─ Circuit breakers
    ├─ Постепенные ликвидации
    └─ Стимулы для keeper ботов

**6. Конкурентный риск**

Проблема: Другие стейблкоины устоялись

Решения::

    ├─ Уникальное value prop (сырьевые товары)

    ├─ Лучшая капитальная эффективность
    ├─ Сильный фокус на сообществе
    └─ Непрерывная инновация
